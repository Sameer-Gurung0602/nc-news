Topics Articles Users

This api allows for users to post articles and comments. It also allows for users to post comments/articles as well as vote and delete them.

Available EndPoints

Endpoint Description Acceptable Queries Request Body Response
/api The homepage of the api, provides details of available endpoints 200- OK
/api/topics Fetches all topics 200- OK - A list of all topics
/api/articles Fetches all articles GET "/api/articles?sort_by=article_id&order=ASC&topic=mitch 200- OK - A list of all topics
/api/users GET Fetches all users 200- OK - A list of all users
/api/articles/:article_id GET Fetches articles by id PATCH updates articles by id Example request body: {inc_votes : 99999} 200- OK - article object
/api/articles/article_id/comments GET - fetches comments by article id POST - adds comment to article Example request body: { "article_id": 2, "body": "I love Cake", "author": "icellusedkars" } 200 -OK - List of comments 201 -CREATED - updated comment
/api/comments/:comment_id DELETES comment by id 204 -no content - empty object
