News Website

Topics Articles Users

This api allows for users to post articles and comments. It also allows for users to post comments/articles as well as vote and delete them.

Available EndPoints

  • /api

    • The homepage of the api, provides an overview of the available endpoints

  • /api/topics

    • GET request to the endpoint fetches all topics

    Responds with 200 OK- A list of all topics

  • /api/articles

    • GET request fetches all articles

    Responds with 200 OK- A list of all articles

    Accepts the following queries: sort_by, order, topic

    Example request GET "/api/articles?sort_by=article_id&order=ASC&topic=mitch

  • /api/users

    • GET request Fetches all users

    Responds with 200 ok- A list of all articles

  • /api/articles/:article_id

    • Fetches articles by id

    Example request: GET api/articles/1

    Responds with 200 ok - article assosciared with article_id 1

    Updates articles by id

    Example request: PATCH api/articles/1

    Example request body: {inc_votes : 99999}

    Responds with 200 ok - updated article object

  • /api/articles/article_id/comments

    • Gets comments by article id

    Responds with 200 ok - list of comments

    Adds comment by article id

    Example request body: { "article_id": 2, "body": "I love Cake", "author": "icellusedkars" }

    Responds with 201 ok - comment object

  • /api/comments/:comment_id

    • Deletes comment by id

    Responds with 204 no content - empty object
