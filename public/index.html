News Website

Topics Articles Users Comments

This api allows for users to post articles and comments. It also allows for users to post comments/articles as well as vote and delete them.

Available EndPoints

  • /api

    • The homepage of the api, provides an overview of the available endpoints

  • /api/topics

    • Fetches all topics

  • /api/articles

    • Fetches all articles

    Accepts the following queries: sort_by, order, topic

  • /api/users

    • Fetches all users

  • /api/articles/:article_id

    • Fetches articles by id

    Updates articles by id

  • /api/articles/article_id/comments

    • Gets comments by article id

    Adds comment by article id

  • /api/comments/:comment_id

    • Fetches comments by id

    Deletes comment by id

  • /api/comments/:comment_id

    • Delete comments by comment id
